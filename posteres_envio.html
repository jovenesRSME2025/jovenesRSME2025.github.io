Información Necesaria para el Envío de Comunicaciones

Para enviar una comunicación, la siguiente información deberar ser provista en el formulario anteriormente indicado:

  1. Información:
    Nombre, apellido(s), afiliación e información de contacto.
  2. Título y resumen en inglés.
  3. Área:
    Deberá indicarse el área de las matemáticas donde se enmarca el trabajo.
  4. Sesiones afines:
    Si procede, deberá indicarse las sesiones en las que el trabajo pudiera encajar.
  5. Códigos MSC:
    Deberá indicarse el código MSC 2020 primario de la temática de la contribución, así como hasta tres códigos segundarios.
  6. Formato:
    Si se desea participar con una charla contribuida, un póster, o cualquiera de las dos opciones a elección del Comité Científico.
  7. Periodo del congreso:
    Se indicará si se prefiere la primera mitad o la segunda mitad para la participación. En caso de pósteres, se podrán seleccionar ambas mitades.

Selección de Comunicaciones

El Comité Científico evaluará las propuestas recibidas hará una selección. Entre las seleccionadas, indicará cuáles se presentarán en formato de charla o póster.

Distribución de Ponencias Contribuidas

Para dar cabida a las ponencias contribuidas, se establecerán sesiones contribuidas para dar cabida a este tipo de charlas, según el Comité Científico considere. Adicionalmente, si lo considera oportuno, el Comité Científico podrá proponer activamente la adición de una de las ponencias contribuidas a una sesión paralela ya existente.
